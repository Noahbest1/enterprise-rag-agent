# 京东售后服务

## 售后服务种类

京东提供以下售后服务,覆盖商品全生命周期:

1. **7 天无理由退换货**(详见退货政策文档)
2. **质量问题 15 天退换货**:商品存在质量问题,自收货 15 天内可免费换货
3. **质保期内维修**:在品牌方质保期内免费维修,质保期以商品详情页为准
4. **价格保护**(京东保价):自营商品购买后若降价,30 天内可申请退还差价

## 申请售后流程

### 通用流程

```
订单页 → 申请售后 → 选择类型(退货/换货/维修/保价)
     → 填写原因和上传凭证 → 提交申请
     → 客服审核(1-2 工作日)
     → 审核通过 → 执行对应操作
```

### 退货后退款时间

- 京东支付、京东白条:1-3 个工作日
- 银行卡:1-7 个工作日
- 微信/支付宝:1-3 个工作日

### 换货时间

- 顺丰或京东快递上门取件:1-2 个工作日
- 京东仓库收到并质检:1-2 个工作日
- 新商品发出:质检通过后次日发货
- 总计一般 5-7 个工作日完成换货

## 价格保护(京东保价)

### 规则

- 仅限**京东自营**商品
- 购买后 **30 天内**,若同款商品(同商品 ID)在京东平台降价,可申请退还差价
- 同一订单每件商品每次只能申请一次价格保护
- 以下情况不支持保价:
  - 第三方卖家商品
  - 秒杀、闪购、团购等促销活动价
  - 使用优惠券、京豆抵扣后的到手价降低(需原价降价才算)

### 申请方式

1. 打开京东 APP → 我的 → 客户服务 → 价格保护
2. 选择需要保价的订单
3. 系统自动比对当前价格,若符合条件,差价以京券或京豆形式返还

## 商品维修

- 在品牌方质保期内,可通过京东申请送修服务
- 京东提供**上门取件送修**(部分品类)或引导用户前往品牌售后网点
- 非质保期内需要用户承担维修费

## 投诉与维权

- 对商品或服务不满意,可在订单页点击"投诉"
- 京东客服 400-606-5500,24 小时服务
- 通过"京东客服"APP 也可发起在线会话
- 对京东处理结果不满意,可向 12315 消协投诉
